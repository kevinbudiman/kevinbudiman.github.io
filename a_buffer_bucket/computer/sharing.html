sharing

Created Wednesday 01 November 2017

If it's Windows, assuming you're all on the same network (wifi or wired), it's honestly pretty easy.

On each computer you want to share files from:

Start -> Run -> "cmd"
"ipconfig" -> Write down the part labelled "IP address", will look like 192.168.1.5 or so.
"hostname" -> write down the response to that.
Right click a folder or drive you want to share, click Properties, click the Sharing tab, click "Advanced Sharing", check the box labelled "Share this folder", give it a name. Save yourself effort and don't use spaces in that name. Default permissions are fine.

On each machine you want to access shared files:

Right click "My Computer" or "Computer" and choose "Map Network Drive"
Pick a drive letter (doesn't matter which, it won't let you choose one that's already in use). In folder, put "\ip.of.sharing.computer\ShareFolderName" - so, "\192.168.1.5\VacationPictures". Put a check in the boxes labelled "Connect using different credentials" and "Reconnect at logon."
When it asks you for username and password, put in "hostname\username" and the password. So if you're getting files off of MomsPC and her username is Jenny, it'd be "MomsPC\Jenny", and whatever her password is.

There ya go. Now every time you power up, assuming the other computer is powered on, if you go to Computer, you will see an icon called "VacationPictures on MomsPC" that you can double-click and access just like you could anything local to your machine.

Let it be said that Windows 7 Home and up have something called a HomeGroup that's supposed to make all that a little less Greek to people, but I've never used it so I have no idea how to do so. You might Google "How do I set up a Windows homegroup" and see if there's a tutorial.
